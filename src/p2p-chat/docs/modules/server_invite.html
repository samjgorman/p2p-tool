initiateHandshake

  • initiateHandshake(me: Keys, name: string, initiator: boolean, recipient: string, friends: Record<string, FriendMetadata>, friendsPath: string, window: BrowserWindow): Promise<void>
    • Defined in server/invite.ts:36

    initiateHandshake constructs a protocol URL inviteLink, broadcasts an invite on a signalhub channel of its encrypted public key, then listens for a response on this channel generated in acceptHandshake(). It then unseals the encrypted response, and validates whether the response matches the encrypted password it sent in its invite. If successful, the function sends an invite-ack to the peer. It then writes the peer's public key and name to the user's friends.json, then attempts to connect over WebRTC in connect().
